随意记录

hexo博客

创建文章

$ hexo n “文章名” n=new bash中可用notepad “文件名”

$ hexo clean //清理

$ hexo g //生成 g=generate

$ hexo s //启动 s=start

  • 去localhost:4000看

将博客部署到GitHub上

  • 打开github创建仓库 命名一定要符合要求

  • 去blog目录下$ cnpm install --save hexo-deployer-git //下载插件

  • 设置config.yml文件，找到最下面

.yml文件中这些”:”后要跟空格 内容有无引号无所谓

部署到远端

$ hexo d

主题

安装

$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

配置

修改hexo根目录下的 _config.yml ： theme: yilia

之后

$ hexo clean

$ hexo g

$ hexo s

注意所在目录！！！
